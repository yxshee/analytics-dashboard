# Installation & Setup

1. Clone the repo  
   ```sh
   git clone https://github.com/yxshee/admybrand-dashboard.git
   cd admybrand-dashboard
   ```
2. Serve the `code/` folder  
   - Open `code/index.html` directly in the browser  
   - Or run a local HTTP server:
     ```sh
     npx http-server code
     ```
